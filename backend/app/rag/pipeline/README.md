# Pipeline Components

This directory contains the main processing pipelines.

## Components

### retrieval_pipeline.py
Main RAG retrieval pipeline:
- Query processing
- Context retrieval
- Answer generation
- Response ranking
- Result formatting

### ingest.py
Document ingestion pipeline:
- Document processing
- Chunk generation
- Embedding creation
- Vector storage
- Sync management

### access_control.py
Access control system:
- Permission management
- Role-based access
- Document visibility
- User authorization
- Security enforcement
